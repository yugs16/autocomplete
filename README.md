# Automplete
An AutoComplete component using React+Typescript+Vite. Check https://autocomplete-mu.vercel.app/

## Setup
After Clone, in terminal
- Run `cd autocomplete`
- Run `npm i`
- Run `npm run dev`
- Check the application on http://localhost:5173/
